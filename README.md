# TERRAWOOD — WPC E-Commerce Demo

International, multilingual e-commerce + quote-request demo for a WPC (Wood-Plastic Composite) building & decoration manufacturer. Single self-contained `index.html`.

## Features
- **7 design directions** — live theme switcher (colour + typography + mood)
- Full-bleed product-focused hero, real product imagery
- **Live variation configurator** — colour / size / finish / pack → image, price & stock update instantly
- **Area (metraj) calculator** — deck / fence / wall → profiles, packs, estimated cost
- **Get-a-Quote** system with file upload + technical downloads
- Multilingual **TR / EN** + multi-currency **TRY / USD / EUR / GBP**
- Responsive, no build step

## Run
Open `index.html` in a browser (internet needed for Google Fonts & Unsplash images).
